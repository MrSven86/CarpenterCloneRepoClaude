# CarpenterCloneRepoClaude

Velocity Systems AB — Carpenter website assets & templates.

## Structure

```
assets/
├── icons/
│   ├── badges-awards/     # Achievement badges, stars, medals
│   ├── best-price/        # Price tags, deals, value icons
│   ├── call-contact/      # Phone, contact, support icons
│   ├── checkbox/          # Checkmark/checkbox variants
│   ├── email-fast/        # Email, fast delivery, mail icons
│   ├── experience/        # Experience, expertise, credibility
│   ├── hand-signals/      # Thumbs up, handshake, likes
│   ├── licensed/          # Certificates, licenses, approvals
│   ├── location/          # Maps, pins, GPS icons
│   ├── mail/              # Email/mail variants
│   ├── materials/         # Wood, metal, construction materials
│   ├── misc/              # Cleaning, documents, other
│   ├── painting-tools/    # Brushes, rollers, buckets, paint
│   ├── phone-24-7/        # 24/7 service, support icons
│   ├── quality/           # Quality badges, assurance
│   ├── quick-time/        # Clock, timer, fast service
│   ├── ratings/           # BBB, A+, grade icons
│   ├── reliable/          # Reliability, trust shields
│   ├── scissors-barber/   # Scissors, barber tools
│   ├── service/           # Service, repair, support
│   ├── shaver/            # Electric razors, trimmers
│   ├── special-business/  # Company, office, business
│   ├── specialist/        # Expert, professional icons
│   ├── trusted/           # Trust, reviews, relationships
│   └── yelp-social/       # Yelp, social media icons
├── badges/                # Satisfaction badges, Google, BBB, logos
├── people/                # Owner/person placeholder SVGs
├── shapes/                # Decorative shapes (white bottom, layers)
└── specific/              # Trade-specific icons (e.g. 50 years)

templates/                 # Carpenter website templates (coming next)
```

## Usage

All assets are accessible via raw GitHub URL:
```
https://raw.githubusercontent.com/MrSven86/CarpenterCloneRepoClaude/main/assets/icons/quality/high-quality.svg
```

## Workflow

1. Clone template for new client
2. Swap company name, colors, phone, photos
3. Push to GitHub → Lovable auto-deploys preview
